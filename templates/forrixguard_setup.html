ForrixGuard Configuration Help

This container allows you to manage the ForrixGuard.json configuration file, which defines the device type, serial number, and connection settings for the system.

  • Upload: Update the device type or settings by uploading a new configuration file. The serial number will be automatically synced with the hardware fuses.
  • Export: Download a backup of the current ForrixGuard.json file to your computer.

Note: The ForrixGuard application service will be automatically restarted when applying a new configuration.

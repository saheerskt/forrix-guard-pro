Dashboard Help
Graph Basics
  • Three Phases: The dashboard displays real-time power data for Phase A, B, and C independently.
  • Devices: Each chart aggregates data from Meters, Inverters, and Setpoints connected to that phase.
Tooltips & Data
  • Vertical Crosshair: Hover over the graph to see a vertical line indicating the exact time slot.
  • Filled Box (■): Indicates Actual Data was received from the device at this timestamp.
  • Hollow Box (□): Indicates Repeated Data (no new value received, previous value held).
Controls
  • Pause/Resume: Stop the live update to inspect historical data closer.
  • Tooltip Toggle: Use the checkbox in each graph header to show/hide tooltips individually.
